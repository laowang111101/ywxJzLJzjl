# 家政管理系统 python624

## 项目概述

家政管理系统是一款使用Python+Django+Vue.js开发的软件，旨在提供家政服务的全面管理解决方案。该系统集成了前后台操作，简化了部署流程，可供学习与参考。

## 技术栈

- **后端：** Python + Django
- **前端：** Vue.js

## 功能模块

系统主要包括以下功能模块：

- 用户管理
- 服务项目管理
- 预约排期管理
- 费用结算管理
- 数据报表统计

## 系统角色

- 系统管理员
- 家政服务人员
- 客户

## 运行环境

- Python 3.8+
- Django 3.x
- Node.js 12.0+
- Vue.js 2.x

## 部署步骤

1. 安装Python和Django环境
2. 配置数据库
3. 前端Vue.js项目编译与构建
4. 部署至Web服务器

## 目录结构

```
家政管理系统/
├── backend/               # 后端代码目录
│   ├── apps/              # 应用代码目录
│   ├── db/                # 数据库相关文件
│   ├── media/             # 媒体文件目录
│   ├── static/            # 静态文件目录
│   ├── manage.py          # 管理脚本
│   └── ...
└── frontend/              # 前端代码目录
    ├── src/               # Vue.js源码目录
    ├── public/            # 公共资源目录
    ├── dist/              # 编译后输出目录
    └── ...
```

## 核心亮点

- **前后端分离：** 前端采用Vue.js，后端采用Django，便于维护与扩展。
- **丰富的功能：** 覆盖家政服务业务全流程，提升管理效率。
